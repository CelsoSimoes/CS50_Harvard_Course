Especificações da Página:

    O website precisa ter
  1. Ao menos 4 Pags HTML, sendo possível ir de uma página para outra, de qualquer uma delas
  2. Ao menos 1 de cada item: Lista, Tabela e Imagem
  3. Ao menos um arquivo com stylesheet
  4. Nos stylesheets, é preciso ter ao menos 5 propriedades CSS(trivial), 5 seletores, e ao menos 1 id e 1 classe.
  5. O stylesheets precisa ser responsivo para Clulares e ter uma query responsiva tambem
  6. Preciso usar Bootstrap 4, ao menos um componente e 2 colunas usando o grid model
  7. Ao menos uma variavel SCSS, com heranca e aninhamento
  8. O README precisa ser completo, contendo todas as informações de cada arquivo, e gerais.
